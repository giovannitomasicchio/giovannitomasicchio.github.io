Inserimento dei dati

Cominciamo con un inserimento dati, che segue la sintassi: INSERT INTO nome_tabella SET nome_campo_1 = valore_1, nome_campo_2 = valore_2 oppure: INSERT INTO nome_tabella (nome_campo_1, nome_campo_2) VALUES (valore_1, valore_2) In generale per lanciare una query in uno script PHP dobbiamo semplicemente chiamare la funzione mysql_query() a cui passare come parametro la stringa contenete la query. Nel caso di un inserimento mysql_query() restituisce TRUE o FALSE, a seconda dell’esito dell’operazione.
