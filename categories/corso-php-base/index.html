while A volte serve semplicemente ripetere un gruppo di istruzioni finché una determinata condizione è soddisfatta. Quando tale condizione non è più verificata le iterazioni devono essere interrotte. Questo è proprio quello che fa il ciclo while che si presenta con una sintassi del tipo: while(condizione) { // istruzioni da iterare } dove l’espressione condizione viene valutata all’inizio di ogni iterazione. Se assume il valore true allora sarà eseguito il gruppo di istruzioni da iterare, altrimenti il ciclo verrà interrotto.
